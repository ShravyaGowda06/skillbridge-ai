# SkillBridge AI

Hackathon project for the Sustainability & Social Impact theme.

SkillBridge AI helps students from underserved communities turn their profile, constraints, and goals into career-aligned learning pathways using Google Gemini through server-side API routes.

## Tech Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- React
- Server-side API routes
- Vercel-ready deployment

## Local Development

Create `.env.local`:

```bash
GEMINI_API_KEY=your_gemini_api_key
GEMINI_MODEL=gemini-3.1-flash-lite
```

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
